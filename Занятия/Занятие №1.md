–û–ø–µ—Ä–∞—Ç–æ—Ä—ã —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞:

- –û–ø–µ—Ä–∞—Ç–æ—Ä —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞: `==`
- –û–ø–µ—Ä–∞—Ç–æ—Ä –Ω–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤–∞: `!=`

–ü—Ä–∏–º–µ—Ä—ã —Å—Ä–∞–≤–Ω–µ–Ω–∏—è –º–µ–∂–¥—É int, float –∏ bool:

```python
x = 5
y = 2.5
z = True

# –°—Ä–∞–≤–Ω–µ–Ω–∏–µ –º–µ–∂–¥—É int –∏ float
print(x > y)  # –í—ã–≤–æ–¥: True
print(x == y)  # –í—ã–≤–æ–¥: False
print(x <= y)  # –í—ã–≤–æ–¥: False

# –°—Ä–∞–≤–Ω–µ–Ω–∏–µ –º–µ–∂–¥—É int –∏ bool
print(x == z)  # –í—ã–≤–æ–¥: False
print(x > z)  # –í—ã–≤–æ–¥: True
print(x <= z)  # –í—ã–≤–æ–¥: False

# –°—Ä–∞–≤–Ω–µ–Ω–∏–µ –º–µ–∂–¥—É float –∏ bool
print(y == z)  # –í—ã–≤–æ–¥: False
print(y < z)  # –í—ã–≤–æ–¥: False
print(y >= z)  # –í—ã–≤–æ–¥: True

```

–í —ç—Ç–∏—Ö –ø—Ä–∏–º–µ—Ä–∞—Ö –º—ã —Å—Ä–∞–≤–Ω–∏–≤–∞–µ–º —Ä–∞–∑–Ω—ã–µ —Ç–∏–ø—ã –¥–∞–Ω–Ω—ã—Ö –∏ –ø–æ–ª—É—á–∞–µ–º —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ã True –∏–ª–∏ False –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç —Ä–µ–∑—É–ª—å—Ç–∞—Ç–∞ —Å—Ä–∞–≤–Ω–µ–Ω–∏—è.

## str

## –¢–∏–ø –¥–∞–Ω–Ω—ã—Ö str

–¢–∏–ø –¥–∞–Ω–Ω—ã—Ö str –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π —Å—Ç—Ä–æ–∫–∏, –∫–æ—Ç–æ—Ä—ã–µ –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –¥–ª—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è —Ç–µ–∫—Å—Ç–æ–≤–æ–π –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏. –°—Ç—Ä–æ–∫–∏ –≤ Python –º–æ–≥—É—Ç –±—ã—Ç—å –∑–∞–∫–ª—é—á–µ–Ω—ã –≤ –æ–¥–∏–Ω–∞—Ä–Ω—ã–µ (`'`) –∏–ª–∏ –¥–≤–æ–π–Ω—ã–µ –∫–∞–≤—ã—á–∫–∏ (`"`).

–°—Ç—Ä–æ–∫–∏ –º–æ–≥—É—Ç —Å–æ–¥–µ—Ä–∂–∞—Ç—å –ª—é–±—ã–µ —Å–∏–º–≤–æ–ª—ã, –≤ —Ç–æ–º —á–∏—Å–ª–µ –±—É–∫–≤—ã, —Ü–∏—Ñ—Ä—ã, —Å–∏–º–≤–æ–ª—ã –ø—É–Ω–∫—Ç—É–∞—Ü–∏–∏ –∏ –ø—Ä–æ–±–µ–ª—ã. –û–Ω–∏ –º–æ–≥—É—Ç –±—ã—Ç—å –ª—é–±–æ–π –¥–ª–∏–Ω—ã, –æ—Ç –ø—É—Å—Ç–æ–π —Å—Ç—Ä–æ–∫–∏ –¥–æ –¥–ª–∏–Ω–Ω–æ–≥–æ —Ç–µ–∫—Å—Ç–∞.

–ù–µ–∫–æ—Ç–æ—Ä—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏, –¥–æ—Å—Ç—É–ø–Ω—ã–µ –¥–ª—è —Ç–∏–ø–∞ –¥–∞–Ω–Ω—ã—Ö str, –≤–∫–ª—é—á–∞—é—Ç:

- –û–ø–µ—Ä–∞—Ü–∏–∏ –∫–æ–Ω–∫–∞—Ç–µ–Ω–∞—Ü–∏–∏ (`+`): –°–æ–µ–¥–∏–Ω—è–µ—Ç –¥–≤–µ —Å—Ç—Ä–æ–∫–∏ –≤ –æ–¥–Ω—É.
- –û–ø–µ—Ä–∞—Ü–∏–∏ –ø–æ–≤—Ç–æ—Ä–µ–Ω–∏—è (`*`): –ü–æ–≤—Ç–æ—Ä—è–µ—Ç —Å—Ç—Ä–æ–∫—É –∑–∞–¥–∞–Ω–Ω–æ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Ä–∞–∑.
- –û–ø–µ—Ä–∞—Ü–∏–∏ –∏–Ω–¥–µ–∫—Å–∞—Ü–∏–∏ (`[]`): –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Å–∏–º–≤–æ–ª —Å—Ç—Ä–æ–∫–∏ –ø–æ —É–∫–∞–∑–∞–Ω–Ω–æ–º—É –∏–Ω–¥–µ–∫—Å—É. –ò–Ω–¥–µ–∫—Å–∞—Ü–∏—è –Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è —Å 0.
- –û–ø–µ—Ä–∞—Ü–∏–∏ —Å—Ä–µ–∑–∞ (`[:]`): –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –ø–æ–¥—Å—Ç—Ä–æ–∫—É, —Å–æ—Å—Ç–æ—è—â—É—é –∏–∑ —É–∫–∞–∑–∞–Ω–Ω–æ–≥–æ –¥–∏–∞–ø–∞–∑–æ–Ω–∞ —Å–∏–º–≤–æ–ª–æ–≤.
- –û–ø–µ—Ä–∞—Ü–∏–∏ –ø—Ä–æ–≤–µ—Ä–∫–∏ –≤—Ö–æ–∂–¥–µ–Ω–∏—è (`in`): –ü—Ä–æ–≤–µ—Ä—è–µ—Ç, —Å–æ–¥–µ—Ä–∂–∏—Ç—Å—è –ª–∏ —É–∫–∞–∑–∞–Ω–Ω–∞—è –ø–æ–¥—Å—Ç—Ä–æ–∫–∞ –≤ —Å—Ç—Ä–æ–∫–µ, –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –±—É–ª–µ–≤–æ –∑–Ω–∞—á–µ–Ω–∏–µ.

```python
# –ü—Ä–∏–º–µ—Ä—ã –æ–ø–µ—Ä–∞—Ü–∏–π —Å–æ —Å—Ç—Ä–æ–∫–∞–º–∏

str1 = "Hello"
str2 = "World"

# –ö–æ–Ω–∫–∞—Ç–µ–Ω–∞—Ü–∏—è —Å—Ç—Ä–æ–∫

# –ü–æ–≤—Ç–æ—Ä–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫–∏
repeated_str = str1 * 3  # "HelloHelloHello"

# –ò–Ω–¥–µ–∫—Å–∞—Ü–∏—è —Å—Ç—Ä–æ–∫–∏
first_char = str1[0]  # "H"

# –°—Ä–µ–∑ —Å—Ç—Ä–æ–∫–∏
substring = str1[1:4]  # "ell"

# –ü—Ä–æ–≤–µ—Ä–∫–∞ –≤—Ö–æ–∂–¥–µ–Ω–∏—è –ø–æ–¥—Å—Ç—Ä–æ–∫–∏
contains_o = "o" in str1  # True
contains_z = "z" in str1  # False

print(hello_world)
print(repeated_str)
print(first_char)
print(substring)
print(contains_o)
print(contains_z)

```

–°—Ç—Ä–æ–∫–∏ –≤ Python —è–≤–ª—è—é—Ç—Å—è –Ω–µ–∏–∑–º–µ–Ω—è–µ–º—ã–º–∏, —á—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –ø–æ—Å–ª–µ —Å–æ–∑–¥–∞–Ω–∏—è —Å—Ç—Ä–æ–∫–∏ –µ–µ –Ω–µ–ª—å–∑—è –∏–∑–º–µ–Ω–∏—Ç—å. –û–¥–Ω–∞–∫–æ, –º–æ–∂–Ω–æ —Å–æ–∑–¥–∞—Ç—å –Ω–æ–≤—É—é —Å—Ç—Ä–æ–∫—É, –∏—Å–ø–æ–ª—å–∑—É—è –æ–ø–µ—Ä–∞—Ü–∏–∏ –Ω–∞–¥ —Å—É—â–µ—Å—Ç–≤—É—é—â–∏–º–∏ —Å—Ç—Ä–æ–∫–∞–º–∏.

–¢–∏–ø –¥–∞–Ω–Ω—ã—Ö str –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–æ –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã—Ö —Ñ—É–Ω–∫—Ü–∏–π –∏ –º–µ—Ç–æ–¥–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ –ø–æ–∑–≤–æ–ª—è—é—Ç –≤—ã–ø–æ–ª–Ω—è—Ç—å —Ä–∞–∑–ª–∏—á–Ω—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ —Å–æ —Å—Ç—Ä–æ–∫–∞–º–∏, —Ç–∞–∫–∏–µ –∫–∞–∫ –∏–∑–º–µ–Ω–µ–Ω–∏–µ —Ä–µ–≥–∏—Å—Ç—Ä–∞, —Ä–∞–∑–¥–µ–ª–µ–Ω–∏–µ –∏ –æ–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫, –ø–æ–∏—Å–∫ –ø–æ–¥—Å—Ç—Ä–æ–∫ –∏ –º–Ω–æ–≥–æ–µ –¥—Ä—É–≥–æ–µ.

```python
# –ü—Ä–∏–º–µ—Ä—ã –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã—Ö —Ñ—É–Ω–∫—Ü–∏–π –∏ –º–µ—Ç–æ–¥–æ–≤ –¥–ª—è —Å—Ç—Ä–æ–∫

str1 = "Hello, World!"

# –î–ª–∏–Ω–∞ —Å—Ç—Ä–æ–∫–∏

# –ò–∑–º–µ–Ω–µ–Ω–∏–µ —Ä–µ–≥–∏—Å—Ç—Ä–∞

# –†–∞–∑–¥–µ–ª–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫–∏
split_str = str1.split(",")  # ["Hello", " World!"]

# –û–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ —Å–ø–∏—Å–∫–∞ —Å—Ç—Ä–æ–∫
joined_str = "-".join(["Hello", "World"])  # "Hello-World"

index = str1.find("World")
```

–í—Å–µ —ç—Ç–æ –º—ã —Ç–æ–∂–µ –±—É–¥–µ–º –∏–∑—É—á–∞—Ç—å –ø–æ —Ö–æ–¥—É –¥–µ–ª–∞.

### –†–∞–±–æ—Ç–∞ —Å —Å—Ä–µ–∑–∞–º–∏ str

–û–ø–µ—Ä–∞—Ü–∏—è —Å—Ä–µ–∑–∞ (`[:]`) –ø–æ–∑–≤–æ–ª—è–µ—Ç –ø–æ–ª—É—á–∏—Ç—å –ø–æ–¥—Å—Ç—Ä–æ–∫—É –∏–∑ —Å—Ç—Ä–æ–∫–∏, –≤—ã–±—Ä–∞–≤ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–π –¥–∏–∞–ø–∞–∑–æ–Ω —Å–∏–º–≤–æ–ª–æ–≤. –°—Ä–µ–∑—ã –≤ Python –∏–º–µ—é—Ç —Å–ª–µ–¥—É—é—â–∏–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å: `—Å—Ç—Ä–æ–∫–∞[–Ω–∞—á–∞–ª–æ:–∫–æ–Ω–µ—Ü:—à–∞–≥]`.

–ü—Ä–∏–º–µ—Ä—ã —Ä–∞–±–æ—Ç—ã —Å —Å—Ä–µ–∑–∞–º–∏:


–í –ø–µ—Ä–≤–æ–º –ø—Ä–∏–º–µ—Ä–µ –º—ã –ø–æ–ª—É—á–∞–µ–º –ø–æ–¥—Å—Ç—Ä–æ–∫—É "World" –∏–∑ —Å—Ç—Ä–æ–∫–∏ "Hello, World!" —Å –ø–æ–º–æ—â—å—é —Å—Ä–µ–∑–∞ `[7:12]`. –ò–Ω–¥–µ–∫—Å—ã —Å–∏–º–≤–æ–ª–æ–≤ –Ω–∞—á–∏–Ω–∞—é—Ç—Å—è —Å 0, –ø–æ—ç—Ç–æ–º—É —Å–∏–º–≤–æ–ª "W" –∏–º–µ–µ—Ç –∏–Ω–¥–µ–∫—Å 7, –∞ —Å–∏–º–≤–æ–ª "d" - –∏–Ω–¥–µ–∫—Å 12.

–í–æ –≤—Ç–æ—Ä–æ–º –ø—Ä–∏–º–µ—Ä–µ –º—ã –ø–æ–ª—É—á–∞–µ–º –ø–æ–¥—Å—Ç—Ä–æ–∫—É "Hello" –∏–∑ —Å—Ç—Ä–æ–∫–∏ "Hello, World!" —Å –ø–æ–º–æ—â—å—é —Å—Ä–µ–∑–∞ `[0:5]`.

–í —Ç—Ä–µ—Ç—å–µ–º –ø—Ä–∏–º–µ—Ä–µ –º—ã –ø–æ–ª—É—á–∞–µ–º –ø–æ–¥—Å—Ç—Ä–æ–∫—É "Hlo ol!" –∏–∑ —Å—Ç—Ä–æ–∫–∏ "Hello, World!" —Å –ø–æ–º–æ—â—å—é —Å—Ä–µ–∑–∞ `[::2]`. –®–∞–≥ 2 –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –º—ã –≤—ã–±–∏—Ä–∞–µ–º –∫–∞–∂–¥—ã–π –≤—Ç–æ—Ä–æ–π —Å–∏–º–≤–æ–ª –∏–∑ —Å—Ç—Ä–æ–∫–∏.

–í —á–µ—Ç–≤–µ—Ä—Ç–æ–º –ø—Ä–∏–º–µ—Ä–µ –º—ã –ø–æ–ª—É—á–∞–µ–º –æ–±—Ä–∞—Ç–Ω—É—é –ø–æ–¥—Å—Ç—Ä–æ–∫—É "!dlroW ,olleH" –∏–∑ —Å—Ç—Ä–æ–∫–∏ "Hello, World!" —Å –ø–æ–º–æ—â—å—é —Å—Ä–µ–∑–∞ `[::-1]`. –®–∞–≥ -1 –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –º—ã –≤—ã–±–∏—Ä–∞–µ–º —Å–∏–º–≤–æ–ª—ã –≤ –æ–±—Ä–∞—Ç–Ω–æ–º –ø–æ—Ä—è–¥–∫–µ.

–≠—Ç–æ –ª–∏—à—å –Ω–µ–∫–æ—Ç–æ—Ä—ã–µ –ø—Ä–∏–º–µ—Ä—ã —Ä–∞–±–æ—Ç—ã —Å —Å—Ä–µ–∑–∞–º–∏ str. –° –ø–æ–º–æ—â—å—é —Å—Ä–µ–∑–æ–≤ –º–æ–∂–Ω–æ –¥–æ—Å—Ç–∞—Ç–æ—á–Ω–æ –≥–∏–±–∫–æ —Ä–∞–±–æ—Ç–∞—Ç—å —Å –ø–æ–¥—Å—Ç—Ä–æ–∫–∞–º–∏ –≤ —Å—Ç—Ä–æ–∫–∞—Ö.

# –ü–µ—Ä–µ–º–µ–Ω–Ω—ã–µ

–ü–µ—Ä–µ–º–µ–Ω–Ω–∞—è - —ç—Ç–æ –∏–º—è, –∫–æ—Ç–æ—Ä–æ–µ —Å–≤—è–∑—ã–≤–∞–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ —Å –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–º —Ç–∏–ø–æ–º –¥–∞–Ω–Ω—ã—Ö. –í Python, –≤—ã –º–æ–∂–µ—Ç–µ —Å–æ–∑–¥–∞–≤–∞—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ –∏ –ø—Ä–∏—Å–≤–∞–∏–≤–∞—Ç—å –∏–º –∑–Ω–∞—á–µ–Ω–∏—è.

–î–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π –≤ Python, –≤—ã –¥–æ–ª–∂–Ω—ã —É–∫–∞–∑–∞—Ç—å –∏–º—è –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π, –∑–∞—Ç–µ–º –ø–æ—Å—Ç–∞–≤–∏—Ç—å –∑–Ω–∞–∫ —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ (`=`) –∏ –ø—Ä–∏—Å–≤–æ–∏—Ç—å –µ–π –∑–Ω–∞—á–µ–Ω–∏–µ. –ù–∞–ø—Ä–∏–º–µ—Ä:

```python
x = 5

```

–í —ç—Ç–æ–º –ø—Ä–∏–º–µ—Ä–µ –º—ã —Å–æ–∑–¥–∞–ª–∏ –¥–≤–µ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ: `x` –∏ `y`. –ü–µ—Ä–µ–º–µ–Ω–Ω–∞—è `x` –∏–º–µ–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ `5`, –∞ –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è `y` –∏–º–µ–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ `"Hello, World!"`.

Python —è–≤–ª—è–µ—Ç—Å—è —è–∑—ã–∫–æ–º —Å –¥–∏–Ω–∞–º–∏—á–µ—Å–∫–æ–π —Ç–∏–ø–∏–∑–∞—Ü–∏–µ–π, —á—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –≤–∞–º –Ω–µ –Ω—É–∂–Ω–æ —É–∫–∞–∑—ã–≤–∞—Ç—å —Ç–∏–ø –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π –ø—Ä–∏ –µ–µ –æ–±—ä—è–≤–ª–µ–Ω–∏–∏. –¢–∏–ø –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ –Ω–∞ –æ—Å–Ω–æ–≤–µ –∑–Ω–∞—á–µ–Ω–∏—è, –∫–æ—Ç–æ—Ä–æ–µ –µ–π –ø—Ä–∏—Å–≤–∞–∏–≤–∞–µ—Ç—Å—è.

–í—ã –º–æ–∂–µ—Ç–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ –≤ –≤—ã—Ä–∞–∂–µ–Ω–∏—è—Ö –∏ –æ–ø–µ—Ä–∞—Ü–∏—è—Ö. –ù–∞–ø—Ä–∏–º–µ—Ä:


# If, elif, else
